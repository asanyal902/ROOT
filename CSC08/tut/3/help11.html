1 Split Level

Run the macro exercise3.C in compiled mode. We, Bertand and Axel, have used that macro to generate yesterday's tree. Note the ROOT file's compression factor. Note the time it takes to write the tree by using the same mechanism as in read_tree.C from yesterday, which we have included in today's sources again for your convenience.

You should run both macros in compiled mode by adding a "+" at the end. gROOT->Time() is an easy way to determine how long things take, see read_tree.C where it's used, too. Just surround the relevant part in exercise3.C by calls to gROOT->Time() such that you get the time it takes to write the ROOT file.

You can access the current file using the global gFile. Check the documentation of TFile to see the function returning the compression factor
