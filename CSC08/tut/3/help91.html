Running a TSelector with PROOF [OPTIONAL!]

Before we can use PROOF you need to start a PROOF daemon from the shell: 

xrootd -c xpd.cf -b
You can now start a PROOF job (a session and a query) by running run.C from within ROOT. Look at run.C to see how similar it is compared to a non-PROOF analysis.

Hint: Look at the lines you never used before!
