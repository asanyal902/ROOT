1 Reading a TTree

Looking at the macro you probably saw that it reads the ROOT file over the network. Usually there is no difference to reading it from a local disk. You can speed it up (even if you were reading from disk) by enabling only the branches that you really need to read. Well, give it a try!

Your the change in the code that selects the branch to read:

...
void read_tree()
{
   // function reading particles collection in a tree

   // open the Root file
   TFile *f = new TFile("/var/www/html/ROOT/exercise2.root");
   // get the tree from the file
   TTree *myTree = (TTree*)f->Get("MyTree");
   // disable all branches 
   myTree->SetBranchStatus("*", 0);
   // enable only the branch we are interested to read
   myTree->SetBranchStatus("muons*", 1);
