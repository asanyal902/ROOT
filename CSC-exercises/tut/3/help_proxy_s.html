2 My First TSelector

We have written a TSelector for you that analyzes this tree; you can find it in TBeamSpotSel.h and TBeamSpotSel.C. Run it in compiled mode, e.g. using TTree::Process() on the tree in beamspot.root. What does it display? 
TChain* ch = new TChain("tree3"); // create the chain, giving the trees' name
ch->Add("beamspot*.root"); // add all files
ch->Process("TBeamSpotSel.C+"); // process the tree with the compiled selector

Looking at the histogram title, the plot shows the x vs. the y vs. the z position of the beam spots. In other words: it shows the 3D distribution of the primary vertexes. You should also look at the code: there are several histogram created in BeginSlave(); one of them is drawn in Terminate(), called hBeamSpotXYZ. It is extracted from the output list so the selector can also work with PROOF.

In Process() you can see that hBeamSpotXYZ is indeed filled with the x, y, and z coordinate of each event's primary vertex.
