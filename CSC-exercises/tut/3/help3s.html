2 PROOF

Instead of using the whole PROOF cluster, only use your local machine by running the selector in proof lite.

You crate a PROOF lite session with TProof::Open("lite"). You tell the chain (let's assume it's called ch) to use PROOF by calling ch->SetProof(). Then simply run ch->Process("TPosCorr.C+").
