Welcome to ROOT's Buildbot!

Buildbot is used to monitor ROOT's code on several platforms. Several sets of monitors are available:

  • Incremental:
    • Recent checkins (only ROOT, only roottest) If you have received an email report about breaking ROOT then this is what you want to check.
    • Status of most recent build (only ROOT, only roottest) Gives a quick overview of which builders currently have problems.
    • History (only ROOT, only roottest) Busy waterfall of builds and events.
  • Hourly (if SVN changed):
    • Recent builds (only ROOT, only roottest) Full build of ROOT (checkout, configure, build) accumulating changes from the previous hour. Only run if SVN was changed during the previous hour.
    • Status of most recent build (only ROOT, only roottest) Gives a quick overview of which builders currently have problems.
    • History (only ROOT, only roottest) Busy waterfall of builds and events.
  • Full builds on request:
    • Recent builds (only ROOT, only roottest)
    • Status of most recent build (only ROOT, only roottest) Gives a quick overview of which builders currently have problems.
    • History (only ROOT, only roottest) Busy waterfall of builds and events.
  • Try builds (on request, using buildbot try with patches not yet checked in):
    • Recent builds (only ROOT, only roottest)
    • Status of most recent build (only ROOT, only roottest) Gives a quick overview of which builders currently have problems.
    • History (only ROOT, only roottest) Busy waterfall of builds and events.
  • All at once:
    • Recent builds
    • Status of most recent build
    • History
    • Transposed grid display with revisions down the side.
    • Recent builder activity
  • Configuration info:
    • Buildslave information
    • ChangeSource information.
