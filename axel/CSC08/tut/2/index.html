Overview

In this session you will learn how to read and write objects, including TTrees.

You need to download the code from 2.tar.bz2 and then run

tar xjf 2.tar.bz2
cd 2

These are your exercises for this session:

  1. I/O

    1. Run the macro ages_io.C For some reason, the histogram does not stay on the canvas. Please repair that!

      HELP!

    2. The script creates a ROOT file. Quit ROOT and restart it, then use the TBrowser to look at the ROOT file. Draw the histogram that's stored in it.

  2. Reading a TTree

    Note that for these exercises, timing is a key ingredient, so please remember the printed times.

    1. Execute read_tree.C+ (note the plus!). Modify the macro to read the muons; use the TBrowser to determine the name of the branch.

      HELP!

    2. Looking at the macro you probably saw that it reads the ROOT file over the network. Usually there is no difference to reading it from a local disk. You can speed it up (even if you were reading from disk) by enabling only the branches that you really need to read. Well, give it a try!

      HELP!

    3. Instead of plotting the Pt distribution, generate a histogram of another value.

      HELP!

    4. Use TTree::Draw() to draw the same expression. Is the performance comparable to the read performance with all branches enabled?
