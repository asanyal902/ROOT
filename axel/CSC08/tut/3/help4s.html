2 My First TSelector

Actually you have more than just one file - the tree is spread over several ROOT files. So you want to create a TChain from them all. How much time of data taking does the tree from all the beamspot*.root files correspond to? 
TChain* ch = new TChain("tree3"); // create the chain, giving the trees' name
ch->Add("beamspot*.root"); // add all files
ch->Draw("eventinfo.fTime"); // draw the date and time. we find 2008-06-23, 16:05.

// there are one million micro seconds in a second,
// so divide them by 1000000. to get fractions of seconds
ch->Draw("eventinfo.fTime.GetSecond() + eventinfo.fMuSec / 1000000.");

But that does not give you enough detail: when was the first event taken? When the last one?

The simplest solution is to scan through the tree, see the documentation of TTree::Scan(): 

ch->Scan("eventinfo.fTime.GetSecond() + eventinfo.fMuSec / 1000000",
                 "", "", 1);

And the last event: 

ch->Scan("eventinfo.fTime.GetSecond() + eventinfo.fMuSec / 1000000.",
                 "", "", 1, ch->GetEntries()-1);

The chain tells you that the first event was taken at 2008-06-23, 16:05:00.76431; the last one was taken at 2008-06-23, 16:05:51.47468.
