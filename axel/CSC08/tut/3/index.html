Overview

In this session you will learn how to analyze trees.

You need to download the code from 3.tar.bz2 and then run

tar xjf 3.tar.bz2
cd 3

  2. PROOF

    1. That worked, but it wasn't impressively fast. What about running the same selector on PROOF? We have set up a PROOF cluster for you - you are sitting in front of it. During the first exercise, you all used one core of your machine. PROOF is configured to use one core of your machine, too - but on all machines! Depending on when you get here the PROOF cluster might be pretty busy or completely empty, and you will get a performance improvement that depends on the available resources.

      So let's run it! The PROOF master is called "server". Any speed difference?

      HELP!

    2. That was fast, right? Now let's do it interactively. Quit ROOT and start it again. Simply run TProof::Open(). The GUI will allow you to do the same as in the previous exercise.
