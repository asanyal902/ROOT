1 My First TSelector

Modify and run the selector to create a histogram of the radius r = sqrt(x^2 * y^2).

Before we were filling a two-dimensional TH2F. The radius is one single value, so we need a one-dimensional histogram TH1F instead. You will need to change both the declaration of the histogram (see the header file TPosCorr.h) and the creation of the object (new TH2F(...)). You only need to pass the value of the radius to TH1F::Fill(...).
