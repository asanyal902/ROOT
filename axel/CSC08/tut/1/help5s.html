3 Drawing and Editing Histograms

  • Save the histogram to styledhist.C (e.g. see the Canvas's file menue, or right-click the histogram) and look at that saved file. Restart ROOT and run styleshist.C to check that ROOT really does store the histogram as C++ code.

    Use the first method previously described (File Menu, Save As)
