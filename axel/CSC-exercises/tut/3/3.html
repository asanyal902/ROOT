ROOT Exercise III

PROOF (1/3)

Instead of using only one core of your computer, let's use them all by running the selector in proof lite.

The lecture tells you how to initialize a local PROOF lite session. Don't forget to tell the TChain to use PROOF for Process(). And yes, that's of course also in the lecture.

You create a PROOF lite session with TProof::Open(""). You tell the chain (let's assume it's called ch) to use PROOF by calling ch->SetProof(). Then simply run ch->Process("TPosCorr.C+").
