1 Reading a TTree

You can speed up the processing by enabling only the branches that you really need to read. Well, give it a try!

Because there are sub-branches in the major branches, you cannot just set the BranchStatus for "muons" - you will have to set it for all the branches under "muons", too. You can use wildcarding for that.
